# PragnaAI
PragnaAI – Early Disease Detection Using Voice & Spectroscopy

Mumbai Hacks 2025 • Team: Tanmeet Kaur • Harsh Kumar Gaurav • Avi Sinha


⭐ Overview

PragnaAI is an AI-powered early-screening system that detects disease before symptoms appear.
For the hackathon, we built a working voice-based Parkinson’s early-risk detector.
We also present our research architecture for cancer detection using photoacoustic spectroscopy.
